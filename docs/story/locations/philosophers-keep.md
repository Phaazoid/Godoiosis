# The Philosopher's Keep

🟡 **Draft canon (open)** · `Story/Locations/Other Notable Locations/The Philosopher's Keep.docx`
**Type:** hidden stronghold · home of [the Philosopher](../characters/npcs/philosopher.md)

Wherever the Philosopher hides has to **carry significance.** The leading idea: it lies **deep below the alchemist nation** — after all, that's where the **ancient society** stood (the Hermetic capital Paracelsus was later built over).

The open question the source poses: **why can no one find it?** Presumably some **powerful alchemy** conceals it. A late-game location (a Part-4 set-piece — see [battle-locales.md](battle-locales.md)).
